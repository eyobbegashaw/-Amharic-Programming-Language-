# Contributing to Amharic Programming Language

First off, thank you for considering contributing to the Amharic Programming Language! It's people like you who will help make this language a powerful tool for the Ethiopian tech community.



## How Can I Contribute?

### 1. Adding Built-in Functions
You can expand the language by adding new native Amharic functions (e.g., math, string manipulation, or date helpers).
* **Where:** Modify the `_built_in_functions` dictionary in `interpreter.py`.
* **Requirement:** Ensure the function names are intuitive in Amharic.

### 2. Improving the Mobile IDE
Help us make the KivyMD app better by:
* Improving the UI/UX.
* Fixing bugs in the terminal output.
* Adding new themes or localization support.

### 3. Syntax Enhancements
Suggest or implement new keywords to make the language more natural to read and write in Amharic.
